# Portfolio_JayzirTech

- 🚀 Professional Portfolio & Pets Gallery
Welcome to my personal portfolio project. This website has been designed to showcase my transition from Mechatronics Engineering to software development, integrating a technological aesthetic with dynamic personal sections.

## 🛠️ Technologies Used
- HTML: Semantic site structure.

- SASS (SCSS): Advanced styling using variables, mixins, animations, and responsive design.

- JavaScript: Interactivity, smooth scrolling, and dynamic menus.

## ✨ Main Features
- Professional Portfolio: Presentation of Python projects and technical skills.

- Pets Gallery: A dedicated page featuring Glassmorphism effects and animated SASS backgrounds.

- Fluid Navigation: Implementation of scroll-into-view for a seamless user experience without reloads.

- Responsive Design: Adaptive menu that transforms into a mobile navigation bar.

- Dynamic Background: Use of radial gradients and infinite animations to simulate technological depth.

3. Open the index.html file in your preferred browser.
